method  ·  surrogate models implemented

A spectrum of surrogates, not a single choice.

The framework ships five regressors that occupy distinct positions on three axes — whether they emit predictive variance, whether they update incrementally, and how their wall-clock scales with N. Each axis decides which acquisition strategies are reachable and how large a problem the model can survive.

01  ·  not adopted

MLP / DNN

Feed-forward / deep nets — considered, rejected

+ universal approximator, scales to high d. − needs N ≫ 10³ to beat GPs (Snoek 2015[pdf]); σ² is a bolt-on (MC-dropout / Laplace / ensembles). SAEA online sits at N=50–500 — out of regime.

form stacked dense layers  ·  σ² via MC-dropout / Laplace
cost retrain 10s–min  ·  data-hungry — wrong regime

02  ·  not adopted

XGBoost

Gradient-boosted trees — considered, rejected

+ fast, robust on tabular small-N, no scaling fuss (Chen 2016[pdf]). − no native σ² (NGBoost / quantile bolt-on); no interpolation property — μ(Xobs) ≠ Yobs breaks NoisyEI baselines. σ² is the load-bearing requirement.

form additive trees  ·  non-differentiable  ·  σ² via NGBoost / quantile
cost ~1 s  ·  cheap — but σ² is bolt-on, not posterior

03  ·  superseded

ELM family

Extreme Learning Machine — Huang 2006[pdf]

+ closed-form one-shot fit; ~ms predict, ~ms retrain — the cheapest backbone in this set, still useful for fallback μ-only mode and pure-prediction sliders. − no native σ² (ensemble proxy collapses past N≈300); superseded once SSGP delivered the same speed with a real posterior.

form cos(W·x + b)  ·  β = pinv(HTH + λI) HTy  ·  ensemble σ²
cost ~ms predict  ·  ~ms retrain  ·  kept as μ-only fallback

04  ·  gp anchor

KRG

Kriging / Gaussian Process — SMT backend

+ rigorous Bayesian posterior with interpolation property (μ(Xobs) ≡ Yobs); the reference σ² in the SAEA literature. − O(N³) Cholesky — ~30 s retrain at N=500; even freezeTheta ablation still pays the cubic. Anchor for accuracy, not for speed.

form Cholesky on K(X, X)  ·  θ via BFGS  ·  freezeTheta ablation
cost O(N³)  ·  ~30 s @ N=500 not real-time

05  ·  primary

SSGP

Sparse Spectrum GP — Lázaro-Gredilla 2010[pdf]

+ Bayesian posterior σ² (better than ensemble proxy); N never enters the cubic term — first fit ~1.5 s, frozen-θ retrain ~ms. Fast enough for per-frame surrogate refresh inside the MOO loop. − M random features hand-tuned (default M=100); no interpolation property → NoisyEI baselines compute against MeanOnObserved, not raw Y.

form cos((x/ℓ)·W + b) RFF  ·  closed-form EM  ·  per-target ARD ℓ  ·  PRESS-LOO τ
cost first fit ~1.5 s frozen-θ retrain ~ms
claim
NN-class is data-hungry; tree-class has no posterior; the ELM family was where the path actually went. What ships is KRG as GP anchor and SSGP as the primary online surrogate — fast enough for per-frame retrain inside the MOO loop, the only model in this set that is.
